1.数据库: 去安装 phpStudy   测试是使用32位的    启动msql 服务器   :使用默认端口

2.数据库表:去安装 Navicat Premium      把 alishow.sql   数据表  导入

3.进入到 package.json 的所在目录下    运行 $ npm install  下载相关依赖包

4.cmd 运行命令    $ node app.js

5.打开浏览器 : 输入 127.0.0.1:8080/index   进入前台首页

	      输入 127.0.0.1:8080/admin/login  进入后台登录页   (账号密码写死了,自接登录)
